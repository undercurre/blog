简单点说，就是开发的时候一大堆工具的集成体，相当于工具箱咯！

好啦！等我地哩组装自己的工具箱啦！

## init a project
选个pnpm做包管理啦梗系！因为一阵要用vue所以选vite做打包工具
```
pnpm create vite
// 然后跟着指引选vue，typescript

// 框架被建出来之后你会获得一个由vue和typescript建成的最初模板（干净的一笔）
pnpm i
pnpm dev
// 出来一个vue的模板页面
```
